How it works

1. Decorative Top Border (CSS ::before)

The .card-header has a ::before pseudo-element positioned absolutely at the top. It uses a linear-gradient that goes from transparent → gold → transparent, creating a glowing line that isn't flat but fades at the sides like a decorative arc. 

.card-header::before {
  content: '';
  position: absolute;
  top: 0;
  left: 10%;
  right: 10%;
  height: 3px;
  background: linear-gradient(
    90deg,
    transparent,
    #ffd700,
    transparent
  );
}

2. 3D Tilt Effect on Hover (JavaScript + CSS)

When the mouse moves over the card, JavaScript calculates how far the cursor is from the center. Then it applies rotateX and rotateY with perspective() to create the illusion that the card bends towards the mouse point. 

// Calculate distance from center
const rotateY = ((x - centerX) / centerX) * 15;
const rotateX = ((centerY - y) / centerY) * 10;

// Apply 3D transformation
el.style.transform =
  `perspective(600px)
   rotateY(${rotateY}deg)
   rotateX(${rotateX}deg)`;

Additionally, a .card-3d-shine overlay shows a radial-gradient that follows the mouse, simulating a light reflection.
